# pixpipe
Pixpipe er en liten, “ekte” bildeplattform der brukeren laster opp et bilde, og systemet prosesserer det asynkront (thumbnail/resize/watermark) via en kø og workers. Du får en jobId, kan følge status, og hente ferdige resultater. Poenget er å lære systembygging + skalering + pipelines + drift.
